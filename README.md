# AI vs Real Image Detection

A deep learning-based image classification system developed to distinguish AI-generated images from real images. This project was developed as a Software Engineering Graduation Project and focuses on detecting synthetic images generated by modern generative AI models.

## 🚀 Live Demo

Try the application online:

**Hugging Face Spaces:**
https://huggingface.co/spaces/Gamzeee/fake-real-image-detector

---

## 📌 Project Overview

With the rapid advancement of generative AI technologies, distinguishing AI-generated images from real photographs has become increasingly challenging. This project aims to automatically detect AI-generated images using transfer learning and fine-tuning techniques.

The system was trained using CIFAKE and GenImage datasets and deployed through a Gradio-based web application.

## ✨ Features

* Binary Image Classification (AI Generated vs Real)

* Transfer Learning
* Fine-Tuning
* Model Comparison (ResNet50, DenseNet121, EfficientNet-B0)

* Gradio Web Interface
