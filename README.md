# AI vs Real Image Detection

A deep learning-based image classification system developed to distinguish AI-generated images from real images. This project was created as a graduation project and focuses on detecting synthetic images generated by modern generative AI models.

## Project Overview

With the rapid advancement of generative AI technologies, distinguishing AI-generated images from real photographs has become increasingly challenging. This project aims to automatically detect AI-generated images using transfer learning and fine-tuning techniques.

The system was trained and evaluated using CIFAKE and GenImage datasets and deployed through a Gradio-based web application.

## Features

* Binary image classification (AI Generated vs Real)
* EfficientNet-B0 architecture
* Transfer Learning
* Fine-Tuning
* Model comparison with ResNet50 and DenseNet121
* Confusion Matrix analysis
* Gradio web interface
* Hugging Face Spaces deployment
* Real-world image testing

## Model Performance

| Model           | Accuracy | Training Time (sec) |
